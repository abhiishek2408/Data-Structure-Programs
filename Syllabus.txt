## 📌 Important Formulas (Interview Ready)

### Distance

```
d = √[(x2 - x1)² + (y2 - y1)²]
```

### Slope

```
m = (y2 - y1) / (x2 - x1)
```

⚠ Handle vertical line separately (`x2 == x1`)

### Collinearity Check (Best Way)

```
(x2 - x1)(y3 - y1) == (y2 - y1)(x3 - x1)
```

✔ Avoids floating-point error
