The **Two Pointer technique** is a very common pattern in DSA (especially arrays and strings). 
It usually uses **two indices that move toward each other or in the same direction** to reduce 
time complexity from **O(n²) → O(n)**.

Below is a **complete roadmap of Two Pointer problems from Basic → Advanced** (commonly asked 
in interviews and competitive programming).

---

# 🎯 Core Two Pointer Patterns

You should recognize **these patterns**:

| Pattern               | Example           |
| --------------------- | ----------------- |
| Opposite direction    | Two Sum Sorted    |
| Same direction        | Remove duplicates |
| Sliding window        | Longest substring |
| Fast & Slow pointer   | Cycle detection   |
| Sorting + two pointer | 3Sum              |
| Multi pointer         | K-Sum             |

---

# 🧠 Golden Rule to Identify Two Pointer

Use two pointers when:

✔ array is **sorted**
✔ need **pair or triplet**
✔ need **subarray / substring**
✔ want **O(n)** instead of **O(n²)**
✔ need **window expansion & shrinking**
